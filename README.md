# OPRA_Datascience
Repo datascience groep 1
RNA‑seq Analyse – OPRA Periode C
Dit repository bevat de eindopdracht voor de RNA‑seq analyse binnen OPRA Periode C. De volledige uitwerking staat in het bestand: rnaseq_opdracht.Rmd

Inhoud van deze repository rnaseq_opdracht.Rmd – het hoofdrapport met alle analyses, code en interpretaties

.gitignore – lijst met uitgesloten bestanden en mappen

README.md – dit document

Alle grote data‑mappen (FASTQ, BAM, indexbestanden, QC‑output en tussenresultaten) zijn bewust uitgesloten via .gitignore, omdat deze bestanden te groot zijn en niet nodig zijn om de opdracht te beoordelen.

Hoe open je het rapport

Clone of download deze repository.

Open het .Rmd‑bestand in RStudio.

Knit het document om de volledige analyse te bekijken.
